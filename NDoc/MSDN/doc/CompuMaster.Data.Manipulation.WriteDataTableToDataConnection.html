Parameters

sourceTable
The table which shall be transferred to the data connection
dataConnection
An opened connection to the data source
ddlLanguage
A DDL language which shall be used for creating/extending a table on the data connection
dropExistingRowsInDestinationTable
If True, all existing rows will be removed first before new rows from the source table will be imported

Remarks

If the table doesn't exist on the data connection, it will be created automatically if supported by the DDL language. Missing columns will be added automatically. In case that a column already exist on the remote database and its datatype doesn't match the datatype in the source table, there might be thrown an exception while data transfer.
